# bup
## 简介
作为一个up主，为了运营好空间，用程序为其助力！


## 安装
1. 基于[conda](https://docs.conda.io/en/latest/miniconda.html)创建虚拟环境
``` shell
conda create -n bup python=3.6
```
2. 切换并激活`bup`虚拟环境
``` shell
conda activate bup
```
3. 安装依赖
``` shell
pip3 install -r requirements.txt
```
4. 运行测试显示usage
``` shell
python3 app.py --help
```
